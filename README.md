## 🚀 Guia de Execução

Para rodar o projeto localmente, siga os passos abaixo:

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/una-meucurriculo-blazor.git

# 2. Acesse a pasta do projeto
cd MeuCurriculo

# 3. Execute o projeto
dotnet run
```

Após executar, abra o navegador no endereço exibido no terminal (geralmente:
👉 https://localhost:5001)

## 🧠 Heurística: Ajuda e Documentação

O sistema foi desenvolvido seguindo o princípio de que deve ser **simples e intuitivo**, dispensando a necessidade de instruções complexas.

* Interface limpa e direta
* Informações organizadas de forma clara (perfil, experiências e objetivo)
* Navegação simples com foco em uma única página
* Componentização com Blazor, facilitando manutenção e entendimento

Mesmo sendo um sistema simples, a documentação (este README) permite que qualquer usuário ou desenvolvedor consiga:

✔ Executar o projeto facilmente
✔ Entender sua estrutura
✔ Modificar ou evoluir a aplicação
